# Quantis design docs

The authoritative design is the Quantis Technical Design Document
(`quantisMVP.pdf`, 27 pages) — architecture, data flow, AI architecture,
risk engine spec, backtesting methodology, roadmap, and appendices with
DB schemas / API endpoints / folder structure.

This repo implements **Phase 1 (MVP)** of Part 16's roadmap. Keep the PDF
here (copy from `~/Downloads/quantisMVP.pdf`) so design and code travel
together.
